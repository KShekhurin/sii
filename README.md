# 🚢 Digital Warships: Фреймворк для разработки AI-агентов

## 1. Предназначение
Данный репозиторий содержит учебный фреймворк на Python для разработки интеллектуальных агентов, играющих в «Морской бой». Проект создан специально для студентов, слушающих курс «СИИ», Видновым Максимом. Проект специально не перегружался сложным синтаксисом и конструкциями языка Python для простого старта разработки агентов под конкретную игру.

## 2. Правила игры
Игра «Морской бой» подразумевает наличие у двух игроков своих полей размером 10 на 10 клеток. Каждый игрок сначала расставляет свои корабли. Длины кораблей на их количество: 4x1, 3x2, 2x3, 1x4. После этого, игроки пытаются уничтожить корабли противника. Побеждает игрок, уничтоживший все корабли противника.
## 3. Структура программы
**Программа поделена на две части**:
STAND - отвечает за вывод результатов сыгранных игр в консоль;
CTRL - отвечает за контроль агентов, хода игры, проверки корректности хода, передачи хода, проверки условия победы; 
**Студентам не нужно изменять эти функции**.

## 4. Подробная инструкция по написанию собственных агентов
**Агент - это функция. Вот как выглядит пример агента:**
``` 
def Pexample(state, last_result):
    if state['phase'] == 'placement': 
        return random_placement() # определенная в коде вспомогательная функция случайной расстановки
    else:
        available = [(r,c) for r in range(10) for c in range(10) if (r,c) not in state['shots']]
        return random.choice(available) 
```
где state - словарь вида {'board': list(list(int)), 'shots': set(tuple[int,int]), 'ships': list(dict), 'phase': str}, причем ships - свои корабли, board - состояние доски (инты заданы по правилам - см. вспомогательные константы)
last_result - словарь вида {'status': str, 'coords': tuple[int, int]}. Статус может быть как статусом клетки, в которую в прошлый раз выстрелил агент, а может быть и статусом ошибки, связанной с клеткой.
В фазе расстановки возвращаем list(list(int)) - всю доску.
В фазе стрельбы возвращаем tuple[int, int] - клетку, в которую стреляем.
