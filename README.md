## Описание
Проект создан для проведения практик по *символическому интеллекту* в игровом формате.
Графический сервер для оркестрации модульных игр и агентов для них.

Основной фокус — возможность реализации всех частей проекта пользователя на питоне

## Использование
1. `pip install PySide6 deepcopy`
2. `git clone https://github.com/KShekhurin/sii.git`
3. `cd sii`
4. `python main.py`
